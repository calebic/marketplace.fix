UPDATE 10 — SMARTER REVIEW/RETRY TARGETING

What changed:
- Added review scoring/classification so weak matches are not all treated the same.
- Review reasons are now grouped into categories like Identity conflict, Missing identity, Year conflict, Metadata conflict, Weak evidence, and Value uncertainty.
- Held-for-review decisions now use a smarter assessment instead of a single flat confidence check.
- Review queue entries are ordered by priority so the worst conflicts rise to the top first.
- Retry ordering now prefers reviewed/hinted items with the highest review pressure.
- Diagnostics now show review focus, priority, and conflict summary when available.

What to test:
- obvious clean vehicles should still autofill directly
- messy/conflicting vehicles should land in review more reliably
- review queue should feel more intelligently ordered
- review/retry with make/model/year hints should prioritize those sources earlier
- diagnostics should explain WHY something was held, not just that it was held
