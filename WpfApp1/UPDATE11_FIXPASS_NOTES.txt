Marketplace Fixer Enhanced V1 - Update 11 Fix Pass

This pass focuses on remaining fix-worthy issues from the Update 10 baseline:
- review queue now honors ReviewPriority and stops treating value/population-only gaps as full review blockers
- ignored review items no longer count as active review items through the old NeedsReview path
- value uncertainty is now a first-class review category instead of falling through category ranking
- mod-wide Internet Lookup now stages writes, writes them to disk first, and only then commits the live workspace state
- selected-save now awaits mod-memory persistence instead of fire-and-forget timing
- default RealVehiclePricingService instances now share one cache-backed singleton to reduce stale multi-instance cache behavior
- minor wording cleanup: internal queue rebuild log now says review queue instead of renamer queue
