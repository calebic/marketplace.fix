Marketplace Fixer Enhanced V1.0 - Update 1 workflow core patch

What this patch changes:
- added WorkflowCore.cs with a first-pass workflow/service layer
- added staged inference options (local-only vs full online-assisted)
- bulk autofill now routes through AutoFillWorkflowService
- review/retry affected-source passes now route through the same workflow core
- single-config autofill now runs asynchronously through the workflow core instead of doing heavy inference directly on the UI thread
- added a WorkspaceRefreshCoordinator to defer one big workspace refresh until the end of bulk/retry operations

Intent of this patch:
- safer architecture-first improvement without rewriting the whole app
- reduce UI freezing on single autofill
- reduce unnecessary online work during Auto Fill All by doing local-first inference and only escalating weaker matches
- make retry logic share the same inference/write pipeline as bulk autofill

- please run a local dotnet restore/build on your machine and report any compile/runtime issue for the next hotfix pass
